
#  JobTrack Pro

A job application tracking dashboard that helps users organize and monitor their job search process.

## Features

- Add job applications
- Track application status
- Search applications
- Dashboard analytics
- Local Storage persistence
- Responsive SaaS-style UI

## Application Stages

- Applied
- Interview
- Offer
- Rejected

##  Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
- Local Storage API
